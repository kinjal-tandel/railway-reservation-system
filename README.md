# Railway Reservation System
A responsive Railway Reservation System website built using HTML, CSS, and JavaScript.

## Features
- Train Inquiry
- Ticket Booking
- PNR Status Check
- Train Schedule
- Help & Support Section
- Responsive User Interface

## Project Structure
- index.html → Home Page
- inquiry.html → Train Inquiry
- booking.html → Ticket Booking
- pnr.html → PNR Status
- schedule.html → Train Schedule
- help.html → Help & Support
- style.css → Styling
- script.js → JavaScript Functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
